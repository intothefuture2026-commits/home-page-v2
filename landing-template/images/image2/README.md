# 이미지 폴더 정리 가이드

이미지를 넣을 때 헷갈리지 않도록, 아래 기준으로 폴더를 나눠서 사용하면 됩니다.

## `common/`
공통으로 여러 영역에서 같이 사용하는 이미지 폴더입니다.

넣으면 좋은 이미지:
- 로고 이미지
- 사이드 로고
- 상단 네비게이션 로고
- 아이콘류
- 공통 배경 이미지
- 팝업 관련 공통 이미지

## `hero/`
페이지 첫 화면에 들어가는 대표 비주얼 이미지 폴더입니다.

넣으면 좋은 이미지:
- 메인 배너 이미지
- 첫 화면 배경 이미지
- 대표 조감도
- 메인 홍보용 컷

## `overview/`
사업 개요, 핵심 요약, 단지 소개 초반부에 들어가는 이미지 폴더입니다.

넣으면 좋은 이미지:
- 사업개요 관련 이미지
- 단지 요약 설명용 이미지
- 핵심 포인트 정리용 이미지

## `location/`
입지 설명, 주변 생활권, 지도 관련 이미지를 넣는 폴더입니다.

넣으면 좋은 이미지:
- 위치도
- 지도 이미지
- 교통망 이미지
- 생활 인프라 설명 이미지
- 학군, 상권, 공원 관련 이미지

## `premium/`
프리미엄 포인트나 장점 설명 구간에서 사용하는 이미지 폴더입니다.

넣으면 좋은 이미지:
- 프리미엄 특화 설명 이미지
- 차별화 포인트 소개 이미지
- 상품성 강조 이미지

## `exterior/`
건물 외관과 단지 외부 모습을 보여주는 이미지 폴더입니다.

넣으면 좋은 이미지:
- 건물 외관 투시도
- 단지 전경
- 파사드 이미지
- 야경 이미지

## `community/`
커뮤니티 시설과 부대시설 관련 이미지를 넣는 폴더입니다.

넣으면 좋은 이미지:
- 커뮤니티센터 이미지
- 피트니스, 독서실, 라운지 이미지
- 주민공동시설 이미지
- 부대시설 안내 이미지

## `floorplan/`
평면도와 배치도, 타입별 도면을 넣는 폴더입니다.

넣으면 좋은 이미지:
- 단지 배치도
- 세대 평면도
- 타입별 평면 이미지
- 동호수 배치도

## `interior/`
실내 공간과 인테리어 분위기를 보여주는 이미지 폴더입니다.

넣으면 좋은 이미지:
- 거실 이미지
- 주방 이미지
- 침실 이미지
- 욕실 이미지
- 마감재 관련 이미지

## `view/`
조망권이나 주변 전경 관련 이미지를 넣는 폴더입니다.

넣으면 좋은 이미지:
- 세대 조망 이미지
- 주변 전경 이미지
- 공원뷰, 시티뷰 이미지

## `popup/`
팝업창, 이벤트 배너, 안내창 등에 사용하는 이미지 폴더입니다.

넣으면 좋은 이미지:
- 팝업 배너 이미지
- 분양 일정 안내 이미지
- 이벤트 안내 이미지
- 공지용 이미지
